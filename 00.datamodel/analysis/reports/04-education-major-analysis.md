## [LIMITATION]

1. **학력 미등록 이력서(4.40%)**: 학력 정보가 없는 352,814건은 분석에서 제외되어 모집단 편향 가능.
2. **GPA 자기보고 편향**: GPA는 구직자가 자발적으로 입력하므로 상위 성적 편향(survivorship bias) 높음. 평균 GPA 3.62는 실제 분포 대비 과대추정 가능.
3. **전공명 비정규화**: 동일 전공의 다양한 표기법으로 인해 전공별 실제 인원은 집계치보다 높을 수 있음.
4. **school_code Unknown 계열**: `Unknown:CXXXXXXX` 형태의 미매핑 코드 다수 존재 — 외부 학교 코드 테이블과의 조인 없이는 학교별 집계 완전성 보장 불가.
5. **재학 기간 이상치**: ended_on - started_on 계산 시 0~15년 필터 적용했으나, 입력 오류로 인한 이상치 잔존 가능.
6. **final_education_level 불일치**: 이력서 헤더의 최종학력과 실제 education 레코드 간 불일치는 본 분석에서 HIGH_SCHOOL 기준 약 10%로 집계되나, Report 11의 전체 school_type 기준 정본 집계에서는 **35.6%**(n=7,665,296)로 확인된다. sort_order 기반 최고 학력 추출 방식의 정확도 한계가 있으며, 불일치의 96%는 final_education_level이 실제보다 높게 설정된 케이스이다.
7. **complete_status 단일 분포**: 전체 이력서의 98.28%가 COMPLETED로 분류되어 완성도 지표로서의 변별력이 사실상 없음.
8. **C4 완성도 분석 한계**: complete_status가 COMPLETED/BASIC_INFO_COMPLETED 두 값만 유의미하게 존재하여 학교별 품질 차별화 불가.
