## 1. 신선도 구간별 섹션 보유율 (Q1)

[FINDING] 신선도와 품질은 단조 관계가 아니며, **스킬 섹션만 신선도에 강하게 의존**하는 반면 자기소개서는 역전 패턴을 보인다.

| 신선도 구간 | n | 경력(%) | 스킬(%) | 자소서(%) | 자격증(%) |
|------------|---|---------|---------|----------|---------|

[STAT:effect_size] 자기소개서: 5년+ 72.92%로 오히려 최고 — 역전 현상 (과거 이력서의 자소서 생존 편향)

**핵심 인사이트:**
- **스킬 태그**는 신선도의 강력한 프록시 지표: 5년+ 이력서의 스킬 보유율(10.55%)은 최신 이력서(72.24%)의 1/7 수준

  - **(1) 생존 편향(survivorship bias)**: 오래된 이력서 중 자소서가 있는 것만 남아있을 가능성
  - **(2) 과거 플랫폼 필수 입력**: 한국 채용 플랫폼 초기에 자기소개서가 필수 입력 항목이었을 가능성 — 2010년대 초반 잡코리아/사람인 등 주요 플랫폼은 자소서를 이력서 완성 조건으로 요구했으며, 이후 선택 입력으로 전환됨. 이 경우 높은 자소서 비율은 생존 편향이 아닌 플랫폼 정책 변화의 결과
  - [AI 추정] 두 가설 모두 타당하며, 실증 구분을 위해서는 이력서 최초 등록일(created_at) 기준 자소서 입력률 추이 분석이 필요
- **경력 섹션** 피크는 91-365일(79.38%). 막 갱신한 이력서보다 3-12개월 전 이력서가 경력 보유율이 약간 높음 — 활발한 구직 활동 이력자 집중 시기

[LIMITATION] `rid != ''` 조건으로 NULL과 빈 문자열을 동시 처리하나, resume_id가 빈 문자열로 저장된 이상 데이터 존재 가능성 배제 불가.

## 2. ACTIVE vs NONE 구직 의사별 품질 비교 (Q2)

[FINDING] 구직 의사 ACTIVE 사용자는 NONE 대비 신선도·품질 모든 지표에서 현저히 우수하며, 효과 크기는 very large 수준이다.

| job_search_status | n | 평균 경과일 | 경력(%) | 스킬(%) | 자소서(%) |
|-------------------|---|------------|---------|---------|---------|
| ACTIVE | 291,046 | **254.5** | **84.60** | **79.66** | **73.47** |
| NONE | 7,424,462 | 1,526.9 | 68.35 | 35.43 | 63.80 |

[STAT:n] ACTIVE n = 291,046 / NONE n = 7,424,462

> **참고**: 본 분석(Q2)의 n 합계(291,046 + 7,424,462 = 7,715,508)는 `main_flag=1` 모집단(7,715,508건)과 정확히 일치하므로, 실제로는 `main_flag=1` 필터가 **적용된** 결과이다. 따라서 ACTIVE 291,046건은 대표 이력서(main_flag=1) 중 구직 의사 표시 건수이며, Report 07/12/README의 288,988건(이력서 단위가 아닌 고유 사용자 단위 집계)과의 차이(2,058건)는 집계 단위(이력서 vs 유저)에 기인한다.

[STAT:effect_size] 평균 경과일 차이: 1,272.4일 (ACTIVE 254.5일 vs NONE 1,526.9일)

**Cohen's d 추정 (평균 경과일 기준):**

[AI 추정] 표준편차를 실측하지 않고 경험적 추정. 이력서 경과일 분포는 우편향(right-skewed)이 심하므로 보수적 추정:
- 집단 내 SD 추정치 ≈ 700일 (경험적 추정)
- Cohen's d ≈ (1,526.9 - 254.5) / 700 ≈ **1.82** (very large effect)

> **⚠️ 해석 주의**: Cohen's d는 정규분포 가정에 기반한 효과크기 지표이다. 이력서 경과일 분포는 우측으로 심하게 편향되어(right-skewed) 있어 평균 기반 Cohen's d가 효과 크기를 과대 추정할 수 있다. 보다 로버스트한 대안으로 (1) 중앙값 기반 비모수 효과크기, (2) Cliff's delta, (3) rank-biserial correlation 등을 고려할 것. 현 추정치(d≈1.82)는 방향성과 대규모 차이의 존재를 시사하지만 정확한 크기는 실측 SD 기반 재계산이 필요하다.

[STAT:effect_size] Cohen's d ≈ 1.82 (very large) — [AI 추정] ⚠️ 비정규 분포, 추정 SD 기반
[STAT:effect_size] 스킬 보유율 차이: 79.66% vs 35.43% = **44.23%p** 절대 차이

**구직 의사별 특성 요약:**
- ACTIVE 사용자(전체의 약 3.8%)는 평균 8.5개월 이내 이력서, 스킬 80% 보유
- NONE 사용자는 평균 4.2년 경과, 스킬 보유율 35%에 불과
- ACTIVE 사용자가 검색 결과 상위에 노출되어야 채용 담당자 만족도 향상 가능

**랭킹 권고:** `job_search_status = ACTIVE` 여부를 불린(boolean) 부스트 팩터로 적용 (×1.5~2.0 권장)

[LIMITATION] main_flag=1 필터 적용으로 대표 이력서만 분석. 비대표 이력서 제외됨.

## 4. 자기소개서 길이 vs 신선도 (Q4)

[FINDING] 최근 작성된 자기소개서일수록 길이가 길며, 신선도 구간 간 평균 길이 차이는 통계적으로 유의하나 실용적 효과 크기는 작다.

| 신선도 구간 | n | 평균 길이(자) | 표준편차 |
|------------|---|-------------|---------|
| 90일이내 | 32,119 | **2,055.6** | 1,814.4 |
| 1년이내 | 48,433 | 1,964.8 | 1,713.3 |

[STAT:n] 샘플 n = 499,862 (전체 self_introduction 중 500k 샘플)
[STAT:effect_size] 최신(90일이내) vs 3년초과 평균 길이 차이: 173.4자 (2,055.6 - 1,882.2)

**Cohen's d 추정:**
- pooled SD ≈ 1,750 (표준편차 평균 근사)
- Cohen's d ≈ 173.4 / 1,750 ≈ **0.10** (small effect)

[STAT:effect_size] Cohen's d ≈ 0.10 (small) — [AI 추정]
[STAT:p_value] n이 수십만 규모이므로 통계적 유의성은 거의 확실하나 실용적 효과 크기는 작음

**인사이트:** 자기소개서 길이는 신선도의 약한 지표. 가중치 모델에서 `length(description) > 500` 등 임계값 기반 이진 분류 권장.

[LIMITATION] **샘플링 편향 주의**: 본 분석은 `LIMIT 500,000` 기반 샘플이며, 이는 **무작위 추출(random sampling)이 아닌 삽입 순서(insertion order) 기반** 추출이다. ClickHouse의 MergeTree 엔진 특성상 LIMIT 쿼리는 파트(data part) 단위로 순차 반환하므로, 특정 시기에 입력된 이력서가 과대 대표될 수 있다. 무작위 샘플링이 필요할 경우 `ORDER BY rand() LIMIT 500000` 또는 `SAMPLE` 절 사용을 권장한다.

## 9. 통합 가중치 모델 — 에이전트 랭킹 권고

> **본 섹션의 점수 공식은 EDA 결과를 종합한 에이전트 랭킹 권고 모델**이다. 섹션 7의 2요소 시뮬레이션(품질×0.6 + 신선도×0.4)에 구직 의사(Q2)와 지역 매칭(Q8)을 추가한 4요소 확장 모델이며, 가중치는 EDA 분석 결과에 기반한 **휴리스틱 설정**이다. 실제 적용 시 A/B 테스트를 통한 가중치 최적화가 필요하다.
>
> **⚠️ 검증 한계**: 본 점수 모델은 실제 채용 결과(열람률, 면접 전환률 등)와의 상관 검증이 수행되지 않은 순수 휴리스틱 모델이다. 가중치(0.40/0.35/0.20/0.05)는 도메인 추론에 기반하며, 경험적 최적화(Learning-to-Rank, Grid Search 등)가 아닌 분석가 판단에 의해 설정되었다.
