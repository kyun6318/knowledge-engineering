### 우선 타겟 세그먼트 (90일 활성률 기준)

**1순위 (가장 신선):**
- `job_search_status = 'ACTIVE'` → 47.82% (n=336,186)
- 20대 + BACHELOR/GRADUATE 조합 → 20%+ 활성률

> 주의: 여기서의 n=336,186은 이력서 기준이다. 사용자 기준 ACTIVE 비율/건수와 분리 해석 필요.

**2순위 (중간 신선도):**
- `visibility_type = 'PUBLIC'` + `career_type = 'EXPERIENCED'` 조합
- 서울/경기 + 대졸 이상

**3순위 (신선도 낮음 — 회피 권장):**
- `visibility_type = 'HEADHUNTER_ONLY'` → 5년+ 노후율 47.85%
- `final_education_level = 'NONE'` → 5년+ 노후율 44.82%
- 40대 → 5년+ 노후율 41.20%
- NEW_COMER → 중앙값 1,391일

### 추천 복합 필터 쿼리

```sql
-- 고신선도 경력자 풀 (추천)
WHERE user_updated_at >= now() - INTERVAL 365 DAY
  AND career_type = 'EXPERIENCED'
  AND visibility_type IN ('PUBLIC', 'HEADHUNTER_ONLY')
  AND main_flag = 1
-- 예상: ~800K-1M건 (전체의 10-12%)

-- 적극 구직자 우선 정렬
ORDER BY
  CASE WHEN job_search_status = 'ACTIVE' THEN 0 ELSE 1 END,
  user_updated_at DESC
```
