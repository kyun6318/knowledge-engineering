> 분석일: 2026-03-04 | 데이터 소스: ClickHouse `import_resume_hub_tmp` | 총 이력서: 8,018,110건

## 목적

에이전트가 MCP를 통해 이력서 데이터를 연결할 때, 어떤 필드 조합이 통계적으로 유의미한 연관성을 가지는지 검증한다. 기술 통계(descriptive statistics)를 넘어 **추론 통계(inferential statistics)** -- 가설 검정, 효과 크기(effect size), 다중 비교 보정(Bonferroni correction) -- 를 수행하여 에이전트 MCP 필터 설계에 대한 근거 기반 시사점을 도출한다.

## 1. 카이제곱 독립성 검정 (Chi-Square Test of Independence)

### 1a. career_type x final_education_level

**가설:**
- H0: 경력 유형과 최종 학력은 독립이다
- H1: 경력 유형과 최종 학력은 연관성이 있다

**교차표 (행 비율):**

| career_type | ASSOCIATE | BACHELOR | GRADUATE | HIGH_SCHOOL | NONE |

**검정 결과:**

| 지표 | 값 |
|------|-----|
| Chi-square | 65,064.9 |
| df | 4 |
| p-value | < 1e-300 |
| Cramer's V | 0.0901 (negligible) |
| N | 8,018,110 |

**해석 [AI 추정]:** 통계적으로 유의미하나 Cramer's V = 0.09로 실질적 연관성은 미미하다. 신입(NEW_COMER)은 학사(BACHELOR) 비율이 48.8%로 경력자(41.0%)보다 높고, 전문대(ASSOCIATE) 비율은 낮다. 경력 유형만으로 학력을 예측하는 것은 실용적이지 않다.

2. **자소서 완성률의 지속적 격차**: 경력자(60~73%)와 신입(34~50%) 간 자소서 완성률 격차는 20~28%p로 일관적이다. 자소서는 경력자 이력서 품질의 핵심 차별 요소이다.

3. **전체 품질 향상 추세**: 평균 완성률은 EXP(r=0.976), NEW(r=0.927) 모두 강한 양의 상관을 보여 시간이 지남에 따라 이력서 품질이 개선되고 있다.

4. **2026년 품질 격차 재확대**: 2024-2025년 축소 추세이던 격차가 2026년에 다시 확대되었으나, 이는 2026년 데이터가 아직 불완전하기 때문일 수 있다.

## 6. 에이전트 MCP 필터 설계 시사점 [AI 추정]

### 6.1 효과크기 기반 필터 우선순위

| 순위 | 필드 조합 | 효과크기 | MCP 필터 권장 |
|------|----------|---------|-------------|
| 1 | **career_type -> age** | d=0.73 (medium) | career_type 필터 시 연령 범위 자동 조정 권장 |
| 2 | **education -> visibility** | h=0.35 (small) | visibility 필터와 education 교차 시 학사+ 편향 경고 |
| 3 | **career_type -> skill count** | d=0.24 (small) | 스킬 기반 매칭 시 career_type별 기대값 차등 적용 |
| 4 | **career_type -> ACTIVE rate** | h=0.16 (negligible) | ACTIVE 필터는 독립 적용, 경력유형 교차 불필요 |
| 5 | 나머지 카이제곱 검정 | V<0.10 (negligible) | 교차 필터 불필요, 각 독립 적용 |

### 6.2 필터 설계 원칙

1. **독립 필터 원칙**: 대부분의 범주형 변수 쌍(career_type x gender, career_type x education 등)의 연관성은 negligible(V < 0.10)이므로, 각 필터를 독립적으로 적용해도 교호작용에 의한 정보 손실은 미미하다.

2. **career_type은 핵심 분류 변수**: career_type은 age(d=0.73), skill_count(d=0.24), ACTIVE_rate(h=0.16), 자소서 완성률(20%p+ 격차)과 모두 연관되어 있어, 에이전트의 1차 필터로 가장 적합하다.

3. **신선도 필터 권장**: 2023년 이후 이력서의 평균 품질이 확연히 높으며(스킬 완성률 50%+ vs 이전 10%미만), `updated_at >= 2023` 필터가 데이터 품질을 크게 개선한다.

4. **visibility_type 필터 주의**: PUBLIC 비율이 학력에 따라 66~82%로 차이나므로, `visibility_type=PUBLIC` 필터 적용 시 학사 이상 학력자가 상대적으로 적게 포함되는 편향이 발생할 수 있다.

5. **ACTIVE 필터의 강력한 축소 효과**: 전체의 3.8%만 ACTIVE이므로 이 필터 단독 적용 시 풀이 급격히 줄어든다. 에이전트는 ACTIVE 필터를 "strict mode"로, 미적용을 "broad mode"로 구분하여 제공할 수 있다.

### 6.3 에이전트 필터 체인 권장 구조

```
[1차 분류] career_type (EXPERIENCED / NEW_COMER)
    |
[2차 품질] updated_at >= 2023 (신선도 필터)
    |
[3차 접근성] visibility_type (PUBLIC / ALL)
    |
[4차 상세] education, skill, job_search_status (독립 적용)
```

## 7. 한계 및 주의사항 (Limitations)

1. **대표본 효과 (Large Sample Effect)**: N=800만에서는 아무리 작은 차이도 통계적으로 유의미해진다. p-value < 1e-300인 결과가 대부분이므로, 통계적 유의성보다 효과크기를 기준으로 실질적 의미를 판단해야 한다.

2. **교란 변수 미통제**: 카이제곱 검정은 두 변수 간 연관성만 검정하며, 제3의 변수(연령, 가입 시점 등)에 의한 교란 효과를 통제하지 않는다. 인과관계를 주장할 수 없다.

3. **관찰 데이터 편향**: 이력서 허브 데이터는 자기 선택(self-selection) 편향이 존재한다. 이력서를 등록한 사람만 분석 대상이며, 등록하지 않은 잠재 구직자는 포함되지 않는다.

4. **2026년 데이터 불완전**: 2026년 데이터는 분석 시점(2026-03) 기준 일부만 누적되어 있으므로, 2026년 수치는 확정적이지 않다.

5. **UNKNOWN/NONE 카테고리**: gender=UNKNOWN(7,695건), education=NONE(353,795건) 등 미입력 데이터가 분석에 포함되어 있으며, 이들이 결과에 미치는 영향을 별도 분석하지 않았다.

6. **다중 비교의 보수성**: Bonferroni 보정은 가장 보수적인 방법이다. Holm-Bonferroni나 Benjamini-Hochberg 같은 덜 보수적인 방법도 고려할 수 있으나, 본 분석에서는 모든 검정이 p < 1e-300이므로 보정 방법 선택이 결과에 영향을 주지 않는다.

## 분석 방법론

- **카이제곱 독립성 검정**: 범주형 변수 쌍의 독립성 검정. 효과크기는 Cramer's V (0-1 척도, 0.1 미만=negligible, 0.3 미만=small, 0.5 미만=medium)
- **Cohen's d**: 두 집단의 연속형 변수 평균 차이를 풀링된 표준편차로 나눈 표준화 효과크기 (0.2 미만=negligible, 0.5 미만=small, 0.8 미만=medium)
- **비율 차이 Z-검정**: 두 집단의 비율 차이 검정. 효과크기는 Cohen's h (arcsine transformation 기반)
- **Bonferroni 보정**: 다중 비교 시 가족별 오류율(FWER) 통제. alpha_adjusted = alpha / k (k=검정 횟수)
- **Pearson 상관**: 연도와 완성률의 선형 관계 강도 측정
